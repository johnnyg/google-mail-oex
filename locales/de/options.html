Google Mail Notifier ist eine kleine und schnelle Erweiterung für Opera, welche die Anzahl der ungelesenden Nachrichten in Ihrem Googlemail-Posteingang anzeigt. Durch einen Klick auf den Button können Sie zudem die Titel der Nachrichten anzeigen lassen.

Hinweis: Diese Erweiterung funktioniert nur mit einem "Google Apps"-Konto wenn dieses bereits auf die neue Infrastruktur umgestellt wurde.

Googlemail Bestätigungscode (OAuth)

Google Mail Notifier verwendet das OAuth -Verfahren um Zugang zu Ihren Googlemail-Konto zu erhalten. Dadurch hat diese Erweiterung keinen Zugriff auf Ihr Password oder andere persönliche Daten aus Ihrem Google-Konto. Rufen Sie den Bestätigungscode durch einen Klick auf den Button ab, bestätigen zu den Zugriff und kopieren Sie den Code in das vorgesehene Feld.

Hier können Sie die Berechtigung für diese Erweiterung löschen, der Abruf wird dann nicht mehr funktionieren.
