Google Mail Notifier ist eine kleine und schnelle Erweiterung für pera, welche die Anzahl der ungelesenen Nachrichten eines oder mehrerer Googlemail-Konten anzeigt. Durch einen Klick auf den Button können Sie zudem die Titel der Nachrichten und weitere Informationen anzeigen lassen. Außerdem haben Sie die Möglichkeit, dass Mail-Links (mailto:) direkt mit Googlemail geöffnet werden.

Googlemail Konten

Google Mail Notifier verwendet das OAuth -Verfahren um Zugang zu Ihren Googlemail-Konto zu erhalten. Dadurch hat diese Erweiterung keinen Zugriff auf Ihr Password oder andere persönliche Daten aus Ihrem Google-Konto. Rufen Sie den Bestätigungscode durch einen Klick auf den Button ab, bestätigen zu den Zugriff und kopieren Sie den Code in das vorgesehene Feld.

Hier können Sie die Berechtigung für diese Erweiterung löschen, der Abruf wird danach nicht mehr funktionieren.
